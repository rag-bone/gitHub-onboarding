# Viewing Markdown Files

Markdown (`.md`) files are plain text that renders as formatted documents — headings, lists, links, and more. You're reading one right now.

Here are a few ways to view them nicely.

---

## GitHub (Web)

No setup needed. Any `.md` file in a GitHub repo renders automatically when you click on it. This is the easiest option for most users.

---

## GitHub Mobile

Install the **GitHub** app ([iOS](https://apps.apple.com/us/app/github/id1477376905) / [Android](https://play.google.com/store/apps/details?id=com.github.android)) and browse repos just like on the web — markdown renders the same way.

---

## VS Code

**Visual Studio Code** has built-in markdown preview. Open any `.md` file and press `Ctrl+Shift+V` (Windows) or `Cmd+Shift+V` (Mac) to open a rendered preview side by side.

To also render **Mermaid diagrams** (flowcharts, sequence diagrams, etc.) install the extension:

> **Markdown Preview Mermaid Support** by Matt Bierner
> Search for it in the Extensions panel (`Ctrl+Shift+X`) or find it in the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=bierner.markdown-mermaid).

---

## Obsidian

**Obsidian** is a dedicated markdown editor with a live preview, graph view, and strong plugin support. It's a good choice if you plan to work with a lot of markdown files locally.

Download at [obsidian.md](https://obsidian.md). Open a folder of `.md` files as a "vault" and Obsidian will render them automatically.
